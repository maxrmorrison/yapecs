yapecs.core.configure(config_module: module, config: Optional[pathlib.Path] = None)[source]¶

Parameters

  • config_module – The submodule containing configuration values

  • config – The Python file containing the updated configuration values. If not provided and the --config parameter is a command-line argument, the corresponding argument is used as the configuration file.
